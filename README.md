## 📌 Project Overview
This is a fully responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript** as part of my Frontend Development Internship at CodeAlpha.

The portfolio showcases my skills, projects, resume, and contact information in a clean and modern UI with smooth animations and interactive elements.

---

## 🚀 Features

### ✅ Core Features
- Responsive design (mobile, tablet, desktop)
- About Me section with profile picture
- Skills section
- Projects showcase section
- Contact section with LinkedIn & Email integration
- Clean and structured layout

### ✨ Additional Features
- Animated navigation bar
- Smooth scrolling between sections
- Animated skills boxes
- Animated project cards
- Each project opens in a **new tab**
- Fully functional LinkedIn button
- Clickable email link
- Hover effects and transition animations
